# QB.Report.Tool.Setup
A zip that contains the neccessary .exe and library files for the QB.Report.Tool

To setup the application you will need the following:

1. An active SQL Server
2. Download the attached .sql script and apply it to the SQL Server
3. Unzip the application .zip file to a folder in order to start the programme
4. Once unzip the appsettings.json need to be updated and specifically the connection string so that it targets the correct SQL server
